# UI/UX Rules

- Design for real SDAL admin/member workflows, not generic landing-page polish.
- Keep admin interfaces dense, clear, and operational.
- Avoid broad visual rewrites and unrelated style churn.
- Check localization for Flutter user-facing text.
- Verify mobile text overflow, tap targets, loading/empty/error states, and accessibility labels where relevant.
