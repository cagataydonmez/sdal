# Design System

## Color Palette
Warm, earthy aesthetic with rust/terracotta accent.

**Light Mode:**
- Canvas: #F7F2E9
- Panel: #FFFCF7
- Accent: #B45637 (rust/terracotta)
- Success: #2A6A4C
- Info: #2F6078
- Danger: #A44635
- Foreground: #261B14
- Foreground Muted: #66594C

**Dark Mode:**
- Canvas: #17120F
- Panel: #29211C
- Accent: #E99A73 (warm orange)
- Success: #80C49A
- Info: #8FC8D6
- Danger: #E6907E
- Foreground: #F6EDE2
- Foreground Muted: #CDBCAA

## Spacing & Radius
- Radius: 12px (XS), 14px (SM), 16px (MD), 18px (LG), 20px (XL), 24px (2XL)
- Gutters: 20px standard padding
- Card spacing: 12px between items

## Typography
- Use Flutter Material 3 text theme
- Title Large for major headings
- Body Small for metadata

## Components
- SurfaceCard: elevation with panel background
- SdalNetworkImage: network image with loading/error states
- FilterChip: for RSVP responses
- FilledButton, OutlinedButton: primary and secondary actions
